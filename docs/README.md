## 📊 项目快速概览

### 核心模块
```
pkg/
├── types/          # 数据类型定义（最基础）
├── store/          # 存储系统（核心服务）
├── image/          # 镜像管理（业务逻辑）
├── container/      # 容器管理（业务逻辑）
├── cli/            # 命令行接口（用户界面）
├── performance/    # 性能优化（高级功能）
├── network/        # 网络功能（高级功能）
├── storage/        # 存储驱动（高级功能）
└── cluster/        # 集群管理（高级功能）
```

### 主要功能
- **镜像管理**：pull、list、remove、build、tag
- **容器管理**：run、start、stop、remove、logs、inspect
- **系统管理**：info、prune
- **集群管理**：init、join、leave、node、task、service
- **高级特性**：性能监控、网络配置、存储优化、多节点
